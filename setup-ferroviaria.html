Ferroviária LLGR

Como Rodar o Projeto Localmente

Opção recomendada — Docker

git clone https://github.com/lucaspwalter/ferroviaria-llgr.git cd ferroviaria-llgr docker compose up --build

Acesse http://localhost:8000. Aplicação, MySQL e schema são configurados automaticamente.

Instalação manual rápida

Com PHP 8.2, MySQL e Git instalados:

git clone https://github.com/lucaspwalter/ferroviaria-llgr.git cd ferroviaria-llgr mysql -u root -p -e "CREATE DATABASE ferrorama;" mysql -u root -p ferrorama < database/schema/ferrorama.sql cp .env.example .env php -S localhost:8000 -t .

No Windows PowerShell, use cmd /c "mysql -u root -p ferrorama < database\schema\ferrorama.sql" e Copy-Item .env.example .env.

Edite apenas usuário e senha em .env. Acesse http://localhost:8000.

Pré-requisitos

  • Docker, para instalação recomendada
  • PHP 8.2+, para instalação manual
  • MySQL
  • Git

Acessar no navegador

http://localhost:8000/index.html
