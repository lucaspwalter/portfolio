Job Scraper

Job Scraper — Como Rodar Localmente

Monitor automático de vagas com FastAPI, React, PostgreSQL no Docker e SQLite no modo manual. Inclui workflows feitos para Codex pesquisar, classificar e apoiar candidaturas.

Opção recomendada — Docker

git clone https://github.com/lucaspwalter/job-scraper.git cd job-scraper docker compose up

Acesse http://localhost:5173. Banco, API e frontend são iniciados automaticamente. Email e WhatsApp permanecem opcionais.

Usar com Codex

cd job-scraper codex $setup

Depois use $scrape, $rank, $apply <URL> e $interview. Dados pessoais ficam ignorados pelo Git.

Instalação manual rápida — SQLite

Não exige Docker nem PostgreSQL. Email e WhatsApp são opcionais.

Terminal 1 — API

git clone https://github.com/lucaspwalter/job-scraper.git cd job-scraper/backend cp .env.example .env python3 -m venv .venv source .venv/bin/activate pip install -r requirements.txt python3 seed.py python3 main.py

Se não usar notificações, deixe todas as variáveis SMTP_* e NOTIFY_* vazias no .env.

Terminal 2 — Frontend

cd job-scraper/frontend npm install npm start

Windows: troque python3 por py, ativação por .\.venv\Scripts\Activate.ps1 e cp por Copy-Item.

Pré-requisitos

  • Docker, somente para opção recomendada
  • Python 3.10 ou superior
  • Node.js 20 ou superior
  • Git
  • Conta Gmail com senha de app, somente se quiser notificações por email

Link para criar senha de app: https://myaccount.google.com/apppasswords

Testando as notificações

Após subir o projeto, clique em Rodar agora na interface web. O sistema vai buscar vagas na Remotive e enviar um email para cada vaga nova encontrada. Você pode acompanhar o histórico de vagas na página Jobs.

Notificações WhatsApp (opcional)

As notificações via WhatsApp são opcionais. Para ativar, preencha as variáveis EVOLUTION_API_URL, EVOLUTION_API_KEY, EVOLUTION_INSTANCE e NOTIFY_PHONE no .env.

Documentação: https://doc.evolution-api.com
