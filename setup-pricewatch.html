PriceWatch

Como Rodar Localmente no Linux/Ubuntu e Windows

Escolha Docker ou instalação manual rápida para Linux, macOS e Windows.

Opção recomendada — Docker

git clone https://github.com/lucaspwalter/pricewatch.git cd pricewatch docker compose up --build

Acesse http://localhost:3000. PostgreSQL, backend e frontend são iniciados automaticamente.

Instalação manual rápida

Após instalar Java 17, Maven, Node.js e PostgreSQL, crie banco e usuário.

Linux: acrescente sudo -u postgres antes de cada comando psql. Windows:

psql -U postgres -c "CREATE USER pricewatch WITH PASSWORD 'pricewatch';" psql -U postgres -c "CREATE DATABASE pricewatch OWNER pricewatch;"

Terminal 1 — Backend no Linux/macOS

git clone https://github.com/lucaspwalter/pricewatch.git cd pricewatch export DATABASE_USERNAME=pricewatch DATABASE_PASSWORD=pricewatch export JWT_SECRET=pricewatch-secret-key-local-1234567890 mvn spring-boot:run

Terminal 1 — Backend no Windows PowerShell

git clone https://github.com/lucaspwalter/pricewatch.git cd pricewatch $env:DATABASE_USERNAME="pricewatch" $env:DATABASE_PASSWORD="pricewatch" $env:JWT_SECRET="pricewatch-secret-key-local-1234567890" mvn spring-boot:run

Terminal 2 — Frontend

cd pricewatch/frontend npm install npm run dev

Acesse http://localhost:3000.
