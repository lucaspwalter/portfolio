PriceWatch

Como Rodar o PriceWatch Localmente

Use Docker para iniciar frontend, API, PostgreSQL e Chromium com a configuração correta.

Opção recomendada — Docker

git clone https://github.com/lucaspwalter/pricewatch.git cd pricewatch cp .env.example .env docker compose up --build

Acesse http://localhost:3000. A API fica em http://localhost:8080.

Docker inicia PostgreSQL, backend, frontend e Chromium. Amazon, KaBuM e AliExpress funcionam sem credenciais adicionais.

Mercado Livre — API obrigatória

Crie uma aplicação no portal de desenvolvedores do Mercado Livre e preencha o arquivo .env:

MERCADOLIVRE_ACCESS_TOKEN= MERCADOLIVRE_CLIENT_ID= MERCADOLIVRE_CLIENT_SECRET=

Sem essas credenciais, produtos do Mercado Livre não podem ser monitorados. Nunca publique o arquivo .env.

Parar e reiniciar

docker compose down docker compose up --build

Use docker compose down -v somente se quiser apagar também o banco local.
